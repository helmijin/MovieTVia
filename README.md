# MovieTVia
최신 영화에 대한 정보(평점, 줄거리, 감독, 출시연도, 후기 등)를 볼 수 있다.    또한 게시판 기능을 활용하여 해당 영화에 대한 리뷰를 작성할 수 있다. 
(ref.watchapedia)
# ⏱ 개발 기간
2022.12.19 ~ 2023.01.06 (19일)
